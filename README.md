# Animal-Crossing-MP3-player
Hello and welcome to my new project!

This is an mp3 music player which plays Animal Crossing Music and background music according to the time of the day and the music chosen.
The interface has been created using tkinter and the mp3 handling has been performed using the pygame library.

The code will be updated with comments and details about how to costumize the music choice.

This repository does not include the mp3 files, which should be named according to the following convention:
Music: ./ancr-sountrack/Season/time_Season.mp3
Background noises: ./acnr-soundtrack/White_noises
The mp3 files should be saved into the same folder as the code.

The seasons should be renamed Sunny, Rainy and Snowy. The program is fully costumizable with any collection of mp3 file, as long as they follow the name convention. Future developments will include generalizing the names and the paths, so that any collection of .mp3 files may be used.
